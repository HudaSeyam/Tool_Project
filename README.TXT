The **UCAS Crypto Tool** is developed by an all-female team of Information Security Engineering students using **Java**. The tool integrates various cryptography algorithms and techniques, offering a comprehensive solution for encryption and decryption.

## Features

- Supports multiple cryptographic algorithms
- User-friendly graphical interface
- Secure and efficient encryption techniques

## How to Run the Project

To run the project from the command line, follow these steps:

1. Navigate to the `dist` folder in your terminal.
2. Execute the following command to run the tool:

   ```bash
   java -jar "wizard.jar"
